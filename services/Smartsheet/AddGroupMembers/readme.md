# Add Group Members

Adds one or more members to a group.

## Configuration
- **Group ID**: Group to add members to
- **Member Emails**: Comma-separated list of email addresses
- **Output Variable**: Variable for result

## Example
Member Emails: `user1@example.com,user2@example.com,user3@example.com`

## Notes
- Users receive email notification
- Requires group admin access
- Users must exist in organization
