# Move Sight (Dashboard)

Moves a dashboard to a different location.

## Configuration
- **Sight ID**: Dashboard to move
- **Destination Type**: Where to move (folder, workspace, or home)
- **Destination ID**: ID of folder/workspace (not needed for home)
- **Output Variable**: Variable for move result

## Notes
- Requires owner access
- Moving to home puts dashboard at top level
- Preserves all dashboard settings and shares
