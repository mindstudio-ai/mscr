# Import CSV to Sheet

Imports a CSV file to create a new Smartsheet.

## Configuration
- **File URL or Path**: URL or path to the CSV file
- **Sheet Name**: Name for the new Smartsheet
- **Header Row Index**: Optional row index for headers (0-based)
- **Output Variable**: Variable for imported sheet

## Notes
- Automatically detects delimiters
- All data imported as text initially
- Large files supported
