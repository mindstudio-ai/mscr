# Update Workspace

Updates an existing workspace's name.

## Inputs

- `workspaceId` (string, required): The ID of the workspace to update
- `name` (string, required): New name for the workspace
- `outputVariable` (string, required): Variable to store the updated workspace

## Outputs

Returns the updated workspace information.

## Example

```javascript
{
  "workspaceId": "123456789",
  "name": "Updated Workspace",
  "outputVariable": "updatedWorkspace"
}
```
