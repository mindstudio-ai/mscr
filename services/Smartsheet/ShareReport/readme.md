# Share Report

Shares a report with a user.

## Configuration
- **Report ID**: Report to share
- **Email Address**: User email to share with
- **Access Level**: Permission level (VIEWER, EDITOR, EDITOR_SHARE, ADMIN)
- **Output Variable**: Variable for share result

## Access Levels
- **VIEWER**: Can view only
- **EDITOR**: Can edit report
- **EDITOR_SHARE**: Can edit and share
- **ADMIN**: Full control

## Notes
- User receives email notification
- Requires owner/admin access
