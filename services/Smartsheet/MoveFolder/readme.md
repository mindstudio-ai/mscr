# Move Folder

Moves a folder to a different location.

## Configuration
- **Folder ID**: Folder to move
- **Destination Type**: Where to move (workspace, folder, or home)
- **Destination ID**: ID of destination (not needed for home)
- **Output Variable**: Variable for move result

## Notes
- Moves folder and all contents
- Preserves folder structure and contents
- Requires appropriate permissions at destination
