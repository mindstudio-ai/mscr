# Copy Folder

Creates a copy of a folder and all its contents.

## Configuration
- **Folder ID**: Folder to copy
- **Destination Type**: Where to copy (workspace, folder, or home)
- **Destination ID**: ID of destination (not needed for home)
- **New Name**: Name for the copied folder
- **Output Variable**: Variable for new folder

## Notes
- Copies all contents (sheets, reports, child folders)
- Creates independent copy
- Preserves folder structure
