# Get Attachment

Gets details of a specific attachment.

## Configuration
- **Sheet ID**: The sheet containing the attachment
- **Attachment ID**: The attachment to retrieve
- **Output Variable**: Variable to store attachment details

## Example Response
```json
{
  "id": 7265982973541252,
  "name": "Document.pdf",
  "attachmentType": "FILE",
  "mimeType": "application/pdf",
  "url": "https://..."
}
```
