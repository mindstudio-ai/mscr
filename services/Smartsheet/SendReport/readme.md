# Send Report

Sends the report as a PDF attachment via email to the designated recipients

## Inputs

- **reportId** (required): (Required) reportID of the report being accessed.
- **format** (optional): Body property
- **formatdetails** (optional): Body property
- **ccme** (optional): Body property
- **message** (optional): Body property
- **sendto** (optional): Body property
- **email** (optional): Body property
- **emailValue** (optional): Body property
- **subject** (optional): Body property
- **outputVariable** (required): Name of the variable to store the response

## Output

The connector stores the Smartsheet API response in the specified output variable.
