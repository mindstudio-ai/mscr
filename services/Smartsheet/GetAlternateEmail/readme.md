# Get Alternate Email

Retrieves details of a specific alternate email address for a user.

## Configuration

### Email Information
- **User ID**: The unique identifier of the user
- **Alternate Email ID**: The ID of the alternate email to retrieve

### Output
- **Output Variable**: Name of the variable to store the email details

## Example Response

```json
{
  "id": 1234567890,
  "email": "john.doe.work@example.com",
  "confirmed": true
}
```

## Notes
- System administrators can view any user's alternate emails
- Users can view their own alternate emails
