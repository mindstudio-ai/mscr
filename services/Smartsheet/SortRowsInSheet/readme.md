# Sort Rows In Sheet

Sorts the rows of a sheet, either in ascending or descending order.

## Inputs

- **sheetId** (required): (Required) Sheet Id of the sheet being accessed.
- **includeExclude** (optional): (Optional) Any of the relevant parameters or query parameters listed for [Get Sheet](../../tag/sheets#operation/getSheet).

- **sortcriteria** (optional): Body property
- **outputVariable** (required): Name of the variable to store the response

## Output

The connector stores the Smartsheet API response in the specified output variable.
