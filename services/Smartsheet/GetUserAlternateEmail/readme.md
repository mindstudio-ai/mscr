# Get User Alternate Email

Gets details about a specific alternate email.

## Inputs

- `userId` (string, required): The ID of the user
- `alternateEmailId` (string, required): The ID of the alternate email
- `outputVariable` (string, required): Variable to store the alternate email details

## Outputs

Returns detailed information about the alternate email including its ID, email address, and confirmation status.

## Example

```javascript
{
  "userId": "123456789",
  "alternateEmailId": "987654321",
  "outputVariable": "alternateEmailDetails"
}
```
