# Import XLSX to Sheet

Imports an Excel (.xlsx) file to create a new Smartsheet.

## Configuration
- **File URL or Path**: URL or path to the XLSX file
- **Sheet Name**: Name for the new Smartsheet
- **Header Row Index**: Optional row index for headers (0-based)
- **Output Variable**: Variable for imported sheet

## Notes
- Converts Excel formatting to Smartsheet
- Supports formulas (where compatible)
- Large files may take time to process
