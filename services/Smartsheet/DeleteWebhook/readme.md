# Delete Webhook

Deletes a webhook.

## Inputs

- `webhookId` (string, required): The ID of the webhook to delete
- `outputVariable` (string, required): Variable to store the deletion result

## Outputs

Returns a success message indicating the webhook was deleted.

## Example

```javascript
{
  "webhookId": "123456789",
  "outputVariable": "deletionResult"
}
```
