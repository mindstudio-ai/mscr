# Set Report Publish Status

Sets the publish status of the report and returns the new status, including the URL of any enabled publishing.

## Inputs

- **reportId** (required): (Required) reportID of the report being accessed.
- **readonlyfullaccessibleby** (optional): Body property
- **readonlyfulldefaultview** (optional): Body property
- **readonlyfullenabled** (optional): Body property
- **readonlyfullshowtoolbar** (optional): Body property
- **readonlyfullurl** (optional): Body property
- **outputVariable** (required): Name of the variable to store the response

## Output

The connector stores the Smartsheet API response in the specified output variable.
