# List Image Urls

Posts an array of Image Url objects that can be used to retrieve the specified cell images.

## Inputs

- **imageid** (optional): Body property
- **error** (optional): Body property
- **refid** (optional): Body property
- **errorcode** (optional): Body property
- **message** (optional): Body property
- **height** (optional): Body property
- **url** (optional): Body property
- **width** (optional): Body property
- **imageidValue** (optional): Body property
- **errorValue** (optional): Body property
- **refidValue** (optional): Body property
- **errorcodeValue** (optional): Body property
- **messageValue** (optional): Body property
- **heightValue** (optional): Body property
- **urlValue** (optional): Body property
- **widthValue** (optional): Body property
- **outputVariable** (required): Name of the variable to store the response

## Output

The connector stores the Smartsheet API response in the specified output variable.
