# Send Rows

Sends specific rows via email to recipients.

## Configuration
- **Sheet ID**: Sheet containing rows
- **Row IDs**: Comma-separated row IDs to send
- **Recipient Emails**: Comma-separated recipient emails
- **Subject**: Email subject line
- **Message**: Optional email message
- **Output Variable**: Variable for result

## Notes
- Recipients receive formatted email with row data
- Useful for notifications and approvals
- Recipients don't need Smartsheet access
