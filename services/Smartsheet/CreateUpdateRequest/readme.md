# Create Update Request

Creates a new update request to collect information from users.

## Configuration
- **Sheet ID**: Sheet for the update request
- **Row IDs**: Comma-separated row IDs to request updates for
- **Column IDs**: Comma-separated column IDs to request updates for
- **Recipient Emails**: Comma-separated recipient emails
- **Subject**: Email subject line
- **Message**: Optional message
- **Output Variable**: Variable for created request

## Notes
- Recipients receive email with form to update specified cells
- Can specify which rows and columns
- Recurring requests supported
