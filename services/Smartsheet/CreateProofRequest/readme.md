# Create Proof Request

Creates a new proof request for an attachment.

## Configuration
- **Sheet ID**: Sheet containing the attachment
- **Row ID**: Row containing the attachment
- **Attachment ID**: Attachment to request approval for
- **Approver Emails**: Comma-separated list of approver emails
- **Message**: Optional message to approvers
- **Output Variable**: Variable for created proof request

## Notes
- Approvers receive email notification
- Supports multiple approvers
- Requires Premium plan
