# Get User

Gets details about a specific user.

## Inputs

- `userId` (string, required): The ID of the user to retrieve
- `outputVariable` (string, required): Variable to store the user details

## Outputs

Returns detailed information about the user including their ID, email, name, status, and permissions.

## Example

```javascript
{
  "userId": "123456789",
  "outputVariable": "userDetails"
}
```
