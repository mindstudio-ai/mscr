# Set Sheet Publish

Sets detailed publish status for a sheet.

## Configuration
- **Sheet ID**: Sheet to set publish status for
- **Read Only Lite Enabled**: Enable lite read-only access
- **Read Only Full Enabled**: Enable full read-only access
- **Output Variable**: Variable for result

## Notes
- Lite: Basic view, no attachments
- Full: Complete view with attachments
- Can enable both or neither
