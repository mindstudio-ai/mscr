# Remove Favorites (Bulk)

Removes multiple items from favorites in a single operation.

## Configuration
- **Object Type**: Type of objects to remove (sheet, workspace, etc.)
- **Object IDs**: Comma-separated list of IDs to remove
- **Output Variable**: Variable for result

## Example
- Object Type: sheet
- Object IDs: 123456789,987654321,555666777

## Notes
- More efficient than removing one at a time
- All objects must be same type
- Maximum 100 items per request
