# List Workspaces

Lists all workspaces accessible to the user.

## Inputs

- `includeAll` (boolean, optional): Include all workspace information
- `outputVariable` (string, required): Variable to store the list of workspaces

## Outputs

Returns a paginated list of workspaces with their details.

## Example

```javascript
{
  "includeAll": true,
  "outputVariable": "workspacesList"
}
```
