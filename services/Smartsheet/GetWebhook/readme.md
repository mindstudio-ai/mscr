# Get Webhook

Gets details about a specific webhook.

## Inputs

- `webhookId` (string, required): The ID of the webhook to retrieve
- `outputVariable` (string, required): Variable to store the webhook details

## Outputs

Returns detailed information about the webhook including its configuration, status, and metadata.

## Example

```javascript
{
  "webhookId": "123456789",
  "outputVariable": "webhookDetails"
}
```
