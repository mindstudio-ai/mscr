# Delete Workspace

Deletes a workspace.

## Inputs

- `workspaceId` (string, required): The ID of the workspace to delete
- `outputVariable` (string, required): Variable to store the deletion result

## Outputs

Returns a success message indicating the workspace was deleted.

## Example

```javascript
{
  "workspaceId": "123456789",
  "outputVariable": "deletionResult"
}
```
