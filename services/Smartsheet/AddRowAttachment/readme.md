# Add Row Attachment

Attaches a file or URL to a specific row.

## Configuration
- **Sheet ID**: Sheet containing the row
- **Row ID**: Row to attach to
- **Attachment Type**: FILE or LINK
- **File Path**: Path to file (for FILE type, max 30MB)
- **URL**: URL to attach (for LINK type)
- **Output Variable**: Variable to store attachment info

## Notes
- Requires editor access
- Max file size: 30MB
